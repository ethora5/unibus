# UniBus - Smart Bus Tracking App

A real-time bus tracking mobile application built using Flutter and Firebase.

## Features
- Live bus tracking (GPS)
- ETA calculation
- Driver tracking system
- Student feedback system
- Admin dashboard

## Technologies
- Flutter (Dart)
- Firebase (Auth + Firestore)
- OpenStreetMap

## About
This project was developed as a Final Year Project (FYP) at UNITEN.
